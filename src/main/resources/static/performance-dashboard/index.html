테스트 순서

  1. `docker compose up -d`로 MySQL, Redis, RabbitMQ를 실행한다.
  2. `SPRING_PROFILES_ACTIVE=local`로 애플리케이션을 실행한다.
  3. 작은 값인 `VUs=5`, `Duration=10s`로 먼저 확인한다.
  4. Sync API와 Async MQ를 같은 조건으로 번갈아 실행한다.

수치 보는 법

Sync는 `hold_http_duration p95`와 성공률을 본다. Async는 접수 시간인 `accept p95`와 실제 완료 시간인 `result ready p95`를 나눠서 본다.
